The goal of this chapter is to learn basic concepts about computers through programming in Python. By the end of the chapter, the user should be able to use basic Python like reading data, datatypes, and writing 'for' loops through the Pandas library. If you can do that already, skip to the next chapter. Chapter 0 is optional and is included for syntax reference.
