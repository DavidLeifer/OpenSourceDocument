# Chapter 1 : SciPy and Matplotlib

This chapter is designed to be written by the reader in a Jupyter Notebook or Markdown file to learn about website scraping and further knowledge on SciPy for statistics and Matplotlib for data visualization. It is a condensed version of my graduate thesis written at Southern Illinois University - Carbondale titled "Social media footprints of public perception on energy issues in the conterminous United States" and is available through the school’s website.

The general outline is below:

- Weeks 0-5 would include using a library similar to the former Twitter website's API Tweepy to collect text information.
- Weeks 8-11 would include learning data visualization with Matplotlib.
- Weeks 12-15 would learn basic statistical functions with SciPy.
